Extension for the Java Util Logging Implementation - x4juli

x4juli is a port of main parts of log4j to the J2SE Logging API. The default implementation is incomplete and therefor still not widely used, even when introduced in J2SE 1.4. x4juli provides logmanagers, formatters, handlers, and filters to fill the gap.
The interface and use of the original API has not been changed, you just have to use the x4juli.jar and an appropriate configuration file to benefit from x4juli.
Additionally x4juli delivers a native implementation of the Jakarta Commons Logging (JCL) API and the Simple Logging Facade for Java (SLF4J) API.

Help from everyone is welcome!

x4juli is hosted at BerliOS Developer.
